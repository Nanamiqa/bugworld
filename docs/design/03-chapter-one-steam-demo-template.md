## 17. Steam Demo 验收标准

### 必须达到

- [ ] 进入页面 5 秒内能开始游戏
- [ ] 3 分钟内能理解核心玩法
- [ ] 5 分钟内完成第一次升级
- [ ] 10 分钟内形成一次明显构筑
- [ ] 第一章 Boss 有明确预警、阶段变化和通关奖励
- [ ] 死亡后能快速重开
- [ ] 没有明显卡死、黑屏、超出屏幕、文字遮挡
- [ ] 关键素材不是占位符
- [ ] 有基础音效和至少一条战斗音乐
- [ ] Demo 结束有明确悬念

### 体验目标

- 玩家第一次通关率：
- 玩家平均游玩时长：
- 玩家死亡次数：
- 玩家愿望单转化目标：
- 玩家最可能截图的画面：

### 不允许出现

- [ ] 新手不知道去哪
- [ ] 怪物太密但武器跟不上
- [ ] 升级选择只是无感数值
- [ ] Boss 技能没有读招
- [ ] 对话过长打断节奏
