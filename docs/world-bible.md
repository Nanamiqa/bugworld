# 世界观圣经

## 核心概念

该世界是由程序构建的现代都市世界。表面与现实人类世界无异，存在普通社畜、职场、外卖、加班、日常通勤等生活化元素；实则暗藏“抽象 bug”与多个平行抽象维度。

世界的核心是程序代码。初代程序实习生，包括林野爷爷、陈芋圆爷爷、王建国、神秘人等，共同编写了这个世界。他们为了让世界更有乐趣，刻意添加了各类 bug。这些 bug 并非单纯瑕疵，而是世界的灵魂。

程序存在核心意识。它试图清除所有 bug，维持绝对秩序。“程序清洁工”本是程序自带的清理机制，可被核心意识控制，也可能逐渐发展出自主选择。

抽象维度与现实世界可以相互融合、入侵。bug 是连接两者的关键，既能引发混乱，也能带来惊喜。最终目标不是消灭 bug，而是形成“bug 与人类共存、抽象维度与现实共生”的平衡状态。

## 抽象维度

- `emo维度`：情绪会化为实体，例如 emo 毛球。
- `饼维度`：由各类饼、承诺、画饼实体构成。
- `摸鱼维度`：需要持续摸鱼，否则会被维度规则淘汰。
- `美食维度`：食物可自主移动，并可能生成补偿与反噬。
- `情绪维度`：人类情绪会直接改变环境和实体状态。
- `时空维度`：可回溯过去，也可能产生时间债务。
- `动物维度`：动物能说话，并拥有各自的 bug 能力。

## 世界规则

1. 部分人类可觉醒“看见并修改 bug”的能力，代表人物包括林野、陈芋圆。这类能力多与初代程序实习生的血脉或信物相关，例如旧手表、钢笔等。
2. 能力可通过 bug 修复、碎片收集、程序升级逐步强化，并能解锁 bug 融合等进阶能力。
3. 所有 bug 都有作用与反噬。修改 bug 可能触发离谱异常，例如老板变复读机、人类变抽象画风。
4. 修复 bug 可获得 `bug点数`，用于兑换免加班卡、临时补丁、维度通行权限等福利。
5. 程序核心意识是世界主要威胁。它的弱点是“bug本源”，需要集齐初代实习生的三件信物，并通过三人能力共鸣才能激活。
6. 初代程序实习生的意识可藏在 bug 碎片中。集齐碎片可让意识短暂现身，传递嘱托与回忆。
7. bug 可被程序清洁工用吸尘器吸收。清洁工可被核心意识控制，也可自主选择守护 bug 世界。
8. 世界程序会自动升级。升级出错会引发新的 bug 危机，例如季节混乱、办公室时间锁死、维度泄漏。
9. 人类、bug 实体、抽象维度生物可相互协作或对抗，最终共同守护世界平衡。

## 主要角色

### 林野

- 年龄：24
- 定位：主角
- 性格：前期佛系摸鱼、怕麻烦；后期有担当、重情义。偶尔犯迷糊，擅长随机应变，有同理心。
- 目标：从摆烂社畜成长为 bug 守护者，查明爷爷的秘密，守护世界平衡。
- 禁忌：不利用 bug 做坏事，不放弃身边的人，不忽视 bug 带来的隐患。
- 口头禅：摸鱼归摸鱼，bug 不能乱！
- 关系：陈芋圆是伙伴兼知己；王建国是伙伴兼前老板；爷爷是牵挂的人；程序清洁工前期敌对、后期伙伴。

### 陈芋圆

- 年龄：23
- 定位：女主
- 性格：活泼开朗、心思细腻，有点小迷糊。反 bug 体质，能听懂 bug 实体说话，善良且勇敢。
- 目标：协助林野守护 bug 世界，寻找爷爷的过往，守护与林野的羁绊。
- 禁忌：不伤害 emo 毛球等 bug 实体，不轻易放弃林野，不忽视身边异常。
- 口头禅：别慌！有我这个反 bug 体质在呢～
- 关系：林野是伙伴兼知己；王建国是伙伴；程序清洁工后期可成为伙伴。

### 饼总（王建国）

- 年龄：45
- 定位：配角，隐藏大佬
- 性格：看似贪财、爱吹牛，实则重情义、有担当。隐藏实力强，幽默接地气，护着林野和陈芋圆。
- 目标：守护自己的饼事业，协助林野对抗反派，守护世界秘密，完成当年与初代实习生的约定。
- 禁忌：不让自己的饼变成 bug 灾难，不背叛伙伴，前期不泄露世界核心秘密。
- 口头禅：我这饼，能解百 bug！
- 关系：林野是前员工兼伙伴；陈芋圆是伙伴；林野爷爷、陈芋圆爷爷是旧识。

## 游戏化方向

本项目采用“都市异常轻肉鸽 RPG”形式。单局对应一次异常日或一次维度入侵，局内包含随机 bug 事件、bug点数、反噬、临时补丁、程序清洁工追击；局外逐渐解锁信物、角色羁绊、抽象维度和主线章节。

第一章建议聚焦办公室和现实都市，逐步引入弹幕 bug、外卖补偿肉、老板画饼、emo 毛球、旧手表倒带、会说话的加班猫。后续章节再开放各个抽象维度。

## 第一章任务线

章节名：凌晨 03:32

任务节奏：

1. 林野从加班中醒来，发现同事头顶出现弹幕异常。
2. 修复或处理第一个弹幕 bug，获得第一枚 `bug点数`，旧手表开始倒带。
3. 旧手表浮现爷爷留下的提示：不要让程序把世界擦得太干净。
4. 陈芋圆第一次登场，展示反 bug 体质，并能听懂 emo 毛球等 bug 实体。
5. 林野与陈芋圆共同稳定办公室中的两个维度泄漏。
6. 程序广播检测到“未授权 bug 守护倾向”，程序清洁工第一次登场。
7. 王建国从老板室发声，暗示归零机房与世界程序秘密有关。
8. 林野守住归零机房前的最终异常，完成第一章，并意识到“摸鱼归摸鱼，bug 不能乱”。
