# 变量城夜巡

一个原创的都市异常轻肉鸽网页游戏原型。玩家扮演夜班调试员，在一座由规则引擎维持的现代城市里处理异常数据、选择武器构筑，并在系统关注度升高前稳定失控现场。

## 当前玩法

- 白色办公室场景中探索随机异常，场景包含工位、盆景、打印机、饮水机、会议桌、服务器柜、外卖取餐区等办公元素。
- 开局选择初始武器：回形针弹弓、键盘宏飞弹、修正液喷枪。
- 武器会自动锁定最近的异常实体并发射子弹，且每把武器已有专属特性雏形。
- 怪物会更密集地从办公室边缘涌入，击败后掉落可拾取的 `bug点`。
- 拾取 `bug点` 获得经验与 `bug点数`，经验满后升级并从三个“变量祝福”中选择强化。
- 五章连续夜巡：办公室订单异常、地铁环线卡帧、夜市哈希雨、承诺塔递归、公共规则引擎最终校准。
- 每章有独立任务线、剧情反转、敌人倾向、Boss 名称与战斗台词，通关章节后会保留武器和局内构筑进入下一章。
- 五个章节 Boss：协议骑手·周行、时刻表管理员、索引摊主·老梁备份、承诺塔根节点、公共规则引擎·零号白箱。
- 本地档案会记录最远章节、通关次数、最近构筑、累计击破和处理异常数量。
- HUD 新增章节进度、构筑摘要和击破统计；结算界面会展示本局章节、等级、击破、异常、升级、耗时和构筑。
- 每章 Boss 后会出现“章节信物”三选一奖励，例如冷掉配送单、迟到者车票、加盐名牌、根承诺叶片、白箱例外条款等，用来形成跨章节构筑质变。
- 概念共鸣系统：升级和信物带有整数、浮点、数组、栈、队列、哈希、树、图、时间、承诺等标签，同类概念叠到 2/4 层会触发额外被动奖励。
- 处理异常会获得或消耗 `bug点数`。
- 武器升级参考动作肉鸽的祝福式三选一结构，包含普通、稀有、史诗品质，并扩展出整数锚点、浮点云雾、数组索引器、栈式回弹、队列自动机、哈希弱点表、承诺树分枝、图节点传导等构筑方向。
- 修改 bug 会增加反噬，反噬过高会引来白箱巡检员。
- 乔柚加入后会持续降低反噬；第五章白箱巡检员转为协助单元，会周期性扫描并减速最近异常。
- 最终目标不是关闭规则，而是校准公共规则引擎：让它学会“差异不等于错误”。

## 操作

- `WASD` / 方向键：移动
- `J` / 回车：修复脉冲
- `Space`：短距闪避，消耗 1 点 `bug点数`
- 武器：自动发射，无需额外按键

## 在线试玩

这是纯静态网页游戏，可以直接通过 GitHub Pages 发布。仓库 Pages 建议设置为 `Deploy from a branch`，分支选择 `main`，目录选择 `/root`；发布后入口通常是：

https://nanamiqa.github.io/bugworld/
