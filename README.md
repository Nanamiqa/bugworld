# 世界 Bug 值班室

一个基于《社畜能看见世界bug后摆烂失败》概念方向做的早期网页游戏原型。

## 当前玩法

- 都市办公室场景中探索随机异常。
- 第一章任务线：办公室弹幕异常、旧手表倒带、陈芋圆登场、程序清洁工第一次出现。
- 处理异常会获得或消耗 `bug点数`。
- 修改 bug 会增加反噬，反噬过高会引来程序清洁工。
- 完成第一章最终节点即可结束当前原型的一天。

## 操作

- `WASD` / 方向键：移动
- `J` / 回车：修复脉冲
- `Space`：短距闪避，消耗 1 点 `bug点数`

## 文件

- `index.html`：游戏入口
- `src/styles.css`：界面样式
- `src/main.js`：游戏逻辑
- `src/data/game-data.js`：世界观、角色、事件、补丁数据
- `docs/world-bible.md`：世界观圣经与游戏化方向
